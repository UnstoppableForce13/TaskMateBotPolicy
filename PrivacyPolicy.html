TaskMateBot Privacy Policy

Effective Date: March 1, 2026 — this policy remains in effect until officially replaced or the bot is discontinued.

1. Information We Collect

  • Telegram Data: Username, user ID, and messages you send to the bot.
  • Task Data: Any tasks, reminders, or rules you create for execution.
  • Optional Data for Improvement: Some personal data may be collected to enhance the bot’s performance and user experience (e.g., preferred commands, interaction patterns).

2. How We Use Information

  • To execute tasks you define.
  • To store task rules for proper execution.
  • To improve the bot experience, optimize features, and fix issues.
  • To provide logging or analytics for performance and usage (aggregated where possible).

3. Data Sharing

  • We do not sell or share personal data with third parties.
  • Data is used internally only for improving bot functionality and reliability.

4. Data Retention

  • Task-related and enhancement-related data is stored only as long as necessary.
  • Users can delete stored tasks or request data removal at any time.

5. Security

Standard safeguards protect stored information. Telegram messages are also subject to Telegram’s privacy rules.

6. Children’s Privacy

TaskMateBot is not intended for children under 13.
